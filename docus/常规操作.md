# 创建项目仓库

## 初始化

```
git init
```

## 关联远程仓库

```
git remote add origin url
```

## 添加到缓存区

```
# 添加所有文件
git add .

# 添加某文件
git add 文件名
```

## 提交缓存

```
# message 为提交备注
git commit -m [message]
```

## 同步本地到远程仓库

```
# 查看远程仓库
git remote -v

# 将 branch(本地分支)同步到remote(远程仓库)
git push remote branch
```

## 拉取远程仓库到本地

```
# 从remote(远程仓库)拉取最新代码到branch(本地)
git pull remote branch

# 获取远程仓库中所有的分支到本地
git fetch remote
```



## 查看分支合并图

```
git log --graph
```
